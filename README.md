# CSV Tools CLI

A command line tool to work with csv files.
## Dependencies

- [node](https://nodejs.org/en/) >= 10
- npm >= 6

## Installation

```bash
npm i -g csv-tools-cli
```

## Usage

```bash
csv-tools-cli diff a.csv b.csv [out.csv]
```
